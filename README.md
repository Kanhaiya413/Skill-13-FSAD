# Skill-13-FSAD

## Project Overview

This project is a Full Stack Application developed as part of FSAD.
It includes a Spring Boot backend and a React frontend.

---

## Tech Stack

### Frontend

* React.js
* HTML
* CSS
* JavaScript

### Backend

* Spring Boot
* Java
* REST APIs

---

## Project Structure

```
Skill-13-FSAD/
│
├── backend/        # Spring Boot backend
└── frontend/       # React frontend
```

---

## Setup Instructions

### Clone the Repository

```
git clone https://github.com/Kanhaiya413/Skill-13-FSAD.git
cd Skill-13-FSAD
```

### Run Backend

```
cd backend/exp13
mvn spring-boot:run
```

### Run Frontend

```
cd frontend
npm install
npm start
```

---

## Application Flow

* Frontend sends requests to backend APIs
* Backend processes and returns response
* Frontend displays the data
